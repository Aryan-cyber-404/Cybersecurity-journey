### Linux file system exploration :-

### direstories learned-

#### /home
It is basically user home directory .
It contains personal files,SSH keys,documnets,pictures,downloads etc.
It keeps user data separate from operating system.

#### /var
Variable run time data that changes frequently.
It contains logs,databases,cachefiles,webserver content,mail queues etc.
It is critical for debugging ,can grow very large and often monitored for disk usage .

#### /bin
Essential user command binaries.
it contains core commands needed for basic system operstion.
It is required in recovery/single-user mode, it must exist for the system to boot and function minimally

#### /root
Home directory for the root user
It contains ssh keys, scripts and bashrc files.
Root may need access even if /home is available.
keeps administrator environment isolated .

#### /tmp
These are temporary files.
Applications store short lived data here.
Often cleared on reboot, world-writable directory,used heavily by installers and programs .

#### /usr
User applications and shares system resources.
It contains most installed softwares, libraries ,documentations and shared resourcs.
It is the largest part of most linux systems.
Package managers insttall software here.

#### /etc
It contains sytem-wide  configuration files and settings for the operating system and installed applications.
Almost every important linux service reads configuration from /etc.

**** User files stored in the /home directory..
**** temprary files are stored in /tmp directory.
**** system configuration files are stored in /etc directory.
