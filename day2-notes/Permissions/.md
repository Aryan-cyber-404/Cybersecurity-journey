like this in
"chmod 600" it gives me only read and write  activity
"chmod 400" it gives me only read activity.

### Q. Why is chmod 777 dangerous ?
chnod 777 is dangerous because here users can read, write and execute files... Suppose an outsider writes or edits malicious files here and forwards it . then the user recieves next will be got attacked by that outsider.. so experts or users don't use 777 usually..
